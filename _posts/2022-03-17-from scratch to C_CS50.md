# From Scratch to C
C는 좀 더 전통적이고 글자에 기반한 언어이다. 수십년이 된 오래된 언어이지만 C는 현대의 새로운 언어들*(파이썬 같은)의 바탕인 것은 확실하다. 

1. function : arguments, return values
2. conditionals
3. Boolean expressions
4. loops
5. variables


앞선 강의에서 배웠던 위의 내용에서 더 나아가 프로그래밍 언어의 문법에 대해서 배우게 된다. 프로그래밍 언어는 인간의 언어에 비해 훨씬 적은 단어와 문법으로 이루어져있다. 하지만 인간의 언어는 조금 틀려도 의미가 전달되는 반면, 프로그래밍 언어는 컴퓨터와의 의사소통이므로 매우 정확히 써야한다.
