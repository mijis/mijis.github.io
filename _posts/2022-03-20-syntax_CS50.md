> ## **수업내용**

```c
#include <studio.h>

int main(void)
{
    printf("hello, wordl\n");
}
```
위는 hello, world를 c언어로 나타낸 것이다.
<br/><br/>


> ## **소감**

자바를 배웠기 때문에 printf의 의미를 알아서 hello, world가 나온다는 걸 알지만, 아직 c 언어의 int 개념이 뭔지 정확히 모른다. 정보처리기사 시험을 칠 때도 대충 찍어 맞추기만해서 C 언어라는 새로운 언어를 배울 것이라는 게 너무 신기하다. 왜 그런지 정확히 알 수는 없지만, CS는 신기하게도 뭔가를 하나를 배우면 다른 것을 보는 시야가 점점 넓어지는 게 다른 과목들 보다는 잘 느껴지는 것 같아서 신기하다.
